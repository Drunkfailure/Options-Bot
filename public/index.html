Backtest

Choose indicators to combine (entry when all selected are bullish; exit when any flips bearish). Pick timeframe, budget, and date range. Trades the underlying stock.

Indicators to combine: use the checkboxes in the Indicators section above. Bullish: enter when all bullish (buy calls/long); exit when any bearish (or only on TP/SL if “Exit on signal flip” is off). Bearish: enter when all bearish (buy puts/short); exit when any bullish (or only on TP/SL). Both: trade both directions.
